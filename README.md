## Getting Started

Run the development server:

```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

To run test:
```bash
npm run test
# or for watch mode
npm run test:watch
```
Developed with:

- React
- Next.js
- Graphql
- Node.js
- Chakra-ui
- Jest
